Учебный проект, реализующий простую доску объявлений с рубриками на фреймворке Django. Пользователь может просматривать все объявления, фильтровать их по рубрикам и добавлять новые записи через веб‑форму.
Технологии

    Python 3.9

    Django 4.2.26

    SQLite (по умолчанию)

    HTML, CSS (статические файлы Django)

Функциональность

    Вывод списка всех объявлений на главной странице.

    Фильтрация объявлений по рубрикам (страница рубрики).

    Просмотр информации об объявлении: заголовок, текст, рубрика, дата публикации.

    Добавление нового объявления через форму.

    Административная панель Django для управления рубриками и объявлениями.

    Общий базовый шаблон с шапкой, навигацией и подключением стилей.

    Подключение собственных статических файлов (style.css, фон bg.jpg) для оформления интерфейса.

Структура проекта

    bboard/models.py — модели Rubric и Bb (объявление).

    bboard/admin.py — регистрация моделей и настройка отображения в админ‑панели.

    bboard/views.py — представления: главная страница, страница рубрики, добавление объявления.

    bboard/urls.py — маршруты приложения (index, rubric_bbs, add).

    templates/layout/basic.html — базовый шаблон с навигацией.

    templates/bboard/index.html — список всех объявлений.

    templates/bboard/rubric_bbs.html — объявления выбранной рубрики.

    templates/bboard/add.html — форма добавления объявления.

    static/bboard/style.css — таблица стилей.

    static/bboard/bg.jpg — фон для заголовка.

Навигация

    «Главная» — список всех объявлений.

    «Добавить» — переход к форме добавления объявления.

    Список рубрик — ссылки на страницы объявлений конкретной рубрики.

Статические ресурсы находятся в каталоге static/bboard/.

## Установка и запуск

Клонировать репозиторий:

    git clone (https://github.com/SPOiBD25pzGusevKorobov/LR2-0112-237-Gusev-Korobov)
    cd LR2-0112-237-Gusev-Korobov

Установить зависимости:

    pip install -r requirements.txt

Выполнить миграции базы данных:

    python manage.py makemigrations
    python manage.py migrate

(По желанию) создать суперпользователя для входа в админ‑панель:

    python manage.py createsuperuser

Запустить сервер разработки:

    python manage.py runserver

Открыть в браузере:

    Главная страница: http://127.0.0.1:8000/

    Админ‑панель: http://127.0.0.1:8000/admin/
